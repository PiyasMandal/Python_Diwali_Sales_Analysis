📊 Diwali Sales Analysis Project


📌 Project Overview

This project focuses on exploratory data analysis (EDA) of Diwali sales data to identify customer purchasing behavior, sales trends, and business insights that can help organizations improve marketing strategies and revenue generation during festive seasons.

The analysis is performed using Python and popular data analysis libraries, making it suitable for business analytics and consulting roles.

🎯 Objectives

Analyze sales performance during the Diwali festival

Understand customer demographics and buying patterns

Identify top-performing products and customer segments

Provide actionable insights for business decision-making

🛠️ Tools & Technologies Used

Python

Jupyter Notebook

Pandas – data manipulation

NumPy – numerical analysis

Matplotlib & Seaborn – data visualization

📂 Dataset Description

The dataset contains customer and transaction details such as:

Gender

Age Group

Marital Status

Occupation

Product Category

Purchase Amount

Data cleaning steps included:

Removing null values

Correcting data types

Eliminating unnecessary columns

📈 Key Analysis Performed

Customer distribution by gender and age group

Sales comparison across occupations and marital status

Top-selling product categories

Purchase behavior analysis using visualizations

🔍 Key Insights

Married customers contributed higher total sales

Female customers showed higher purchasing activity during Diwali

Certain occupations and age groups were key revenue drivers

Festive offers can be optimized based on customer segmentation

📊 Visualizations

The project includes:

Bar charts

Count plots

Sales comparison graphs

These visual insights help in quick business interpretation, similar to real-world consulting dashboards.
